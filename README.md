
# Advent of Code (py)

## 2022+
More information at https://adventofcode.com/2022/about

Setup:

```bash
poetry install
```

Starting a new day / work on a day:

```bash
cd 2022
./day 1
```

Running tests:

```bash
cd 2022
poetry run pytest
```

Running solution:

```bash
cd 2022
poetry run python day1.py
```
